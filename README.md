# Suoraan tilalta (Direct from Farm)

A web application for listing and managing farms that sell berries, meat, and vegetables directly to consumers.

**Tech Stack:**
- **Frontend:** React + Vite
- **Backend:** Express.js + SQLite
- **Language:** JavaScript

## Prerequisites
- Node.js (v18+)
- npm

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start the Application**
    This will start both the backend API and the frontend concurrently.
    ```bash
    npm run dev
    ```

3.  **Access the App**
    - **Frontend:** [http://localhost:5173](http://localhost:5173)
    - **Backend API:** [http://localhost:3000](http://localhost:3000)

## Features
- View a list of local farms.
- Add new farms (Name, Description, Location, Products).
- Remove farms from the list.
